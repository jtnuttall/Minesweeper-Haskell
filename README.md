## Haskell Minesweeper

- Minesweeper clone in Haskell

- Uses the Gtk3 binding from Hackage

- To use:
	1. Select a grid size by entering rows and colums into the entries

	2. Press `enter` or `Run`

	3. Play minesweeper

	4. To start a new game, press `Run` again
